# Тарифы

[← Назад к оглавлению](./README.md)

**Base URL:** `https://common-api.wildberries.ru`

---

## Методы

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/v1/tariffs/commission` | Комиссия по категориям |
| GET | `/api/v1/tariffs/box` | Тарифы для коробов (доставка, хранение) |
| GET | `/api/v1/tariffs/pallet` | Тарифы для монопаллет |
| GET | `/api/tariffs/v1/acceptance/coefficients` | Тарифы на поставку (14 дней) |
| GET | `/api/v1/tariffs/return` | Тарифы на возврат |

---

## Комиссия

Метод `/api/v1/tariffs/commission` возвращает комиссию WB по родительским категориям товаров.

---

## Тарифы на доставку и хранение

Метод `/api/v1/tariffs/box` возвращает тарифы для товаров в коробах:
- Стоимость доставки
- Стоимость хранения

---

## Коэффициенты приёмки

Метод `/api/tariffs/v1/acceptance/coefficients` возвращает коэффициенты приёмки для конкретных складов на **14 дней вперёд**.

**Рекомендация:** Выбирайте склады с минимальным коэффициентом — это снижает стоимость приёмки.

---

## Тарифы на возврат

Метод `/api/v1/tariffs/return` возвращает:
- Стоимость перевозки товаров до продавца
- Стоимость обратной логистики возвратов
