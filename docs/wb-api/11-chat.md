# Чат с покупателями

[← Назад к оглавлению](./README.md)

**Base URL:** `https://buyer-chat-api.wildberries.ru`

**Лимит:** 10 запросов/10 сек

---

## Методы

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/v1/seller/chats` | Список всех чатов |
| GET | `/api/v1/seller/events` | События чатов (сообщения) |
| POST | `/api/v1/seller/message` | Отправить сообщение |
| GET | `/api/v1/seller/download/{id}` | Скачать файл из сообщения |

---

## Работа с чатами

### Алгоритм:

1. **Получить список чатов** — `GET /api/v1/seller/chats`
2. **Получить события** — `GET /api/v1/seller/events`
   - Параметр `next` для пагинации
   - Повторять пока `totalEvents > 0`
   - Новые чаты имеют `isNewChat: true`
3. **Отправить ответ** — `POST /api/v1/seller/message`

### Формат отправки:
```json
{
  "chatId": "идентификатор_чата",
  "message": "Текст сообщения"
}
```

---

## Важно

- Чат инициируется только покупателем
- Ответить нужно в течение **10 дней**
- Нельзя игнорировать обращения
- Ответы должны быть по существу и следовать деловому этикету
