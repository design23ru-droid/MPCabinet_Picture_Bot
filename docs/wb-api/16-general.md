# Общие методы

[← Назад к оглавлению](./README.md)

**Base URL:** `https://common-api.wildberries.ru`

---

## Проверка и информация

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/ping` | Проверка подключения |
| GET | `/api/communications/v2/news` | Новости портала продавцов |
| GET | `/api/v1/seller-info` | Информация о продавце |

---

## Управление пользователями

| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | `/api/v1/invite` | Создать приглашение пользователя |
| GET | `/api/v1/users` | Список пользователей |
| PUT | `/api/v1/users/access` | Изменить права доступа |
| DELETE | `/api/v1/user` | Удалить пользователя |

---

## Проверка токена

Метод `GET /ping` позволяет:
- Проверить доставку запроса
- Проверить валидность токена
- Проверить соответствие токена категории API

**Токен** — это JWT согласно RFC 7519. Срок действия: 180 дней.
