## Баланс

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/v1/account/balance` | Баланс продавца |

**Возвращает:**
- Текущий баланс
- Сумма к выводу

**Лимит:** 1 запрос/мин

---

## Отчёты о реализации

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/v5/supplier/reportDetailByPeriod` | Детальный отчёт о реализации |

**Параметры:**
- `dateFrom` — дата начала
- `dateTo` — дата окончания
- `rrdid` — курсор пагинации (ID последней записи)

**Важно:** Отчёт формируется еженедельно.

---

## Документы

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/v1/documents/categories` | Категории документов |
| GET | `/api/v1/documents/list` | Список документов |
| GET | `/api/v1/documents/download` | Скачать документ |
| POST | `/api/v1/documents/download/all` | Скачать несколько документов (до 50) |

**Лимиты:**
- Список документов: 1 запрос/10 сек
- Скачивание одного: 1 запрос/10 сек
- Скачивание нескольких: 1 запрос/5 мин
